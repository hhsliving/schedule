# 가전팀 편성표 뷰어

GitHub Pages에서 바로 볼 수 있는 정적 HTML 편성표 뷰어입니다.

## GitHub Pages로 공개하기

1. 이 변경사항을 GitHub에 병합합니다.
2. GitHub 저장소의 **Settings → Pages**로 이동합니다.
3. **Build and deployment**에서 Source를 **Deploy from a branch**로 선택합니다.
4. Branch는 공개하려는 브랜치와 `/ (root)`를 선택한 뒤 저장합니다.
5. 배포가 끝나면 `https://<계정명>.github.io/<저장소명>/`에서 `index.html`이 열립니다.

## 파일 구성

- `index.html`: 웹에서 열리는 편성표 뷰어입니다.
- `images/`: 편성표 PNG 이미지를 넣는 폴더입니다.

## 조작법

- 사업부 / 팀편성 탭: 시트 전환 (키보드 `1` / `2`)
- `‹` / `›` 버튼: 주 이동 (키보드 `←` / `→`)
- 찾기 드롭다운: 연도 · 월 · 주차 선택해서 바로 이동
- 아래 주차 레일: 클릭해서 바로 이동 (주황 표식 = 월 교차 주간)
- 이미지 클릭: 원본 크기로 확대 (드래그로 이동, 다시 클릭하거나 `ESC`로 축소)

## 주차 추가 방법

1. `images` 폴더에 아래 이름 규칙으로 PNG를 넣습니다.
   - `dept-2026-08-1.png` (사업부)
   - `team-2026-08-1.png` (팀편성)
   - 규칙: `{dept|team}-{연도}-{월2자리}-{주차}.png`
2. `index.html` 상단의 `WEEKS` 배열에 한 줄을 추가합니다.

```js
{ y: 2026, m: 8, w: 1, span: "08/03 ~ 08/09", cross: false },
```

`cross`는 월이 바뀌는 주간이면 `true`로 설정합니다.
