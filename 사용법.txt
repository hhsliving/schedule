편성표 요일 행 위치 통일 패치

포함 파일
  tools/build_sheet.py
  tools/render.py

적용 방법
1. ZIP 압축을 풉니다.
2. GitHub 저장소 https://github.com/hhsliving/schedule 로 이동합니다.
3. 저장소의 tools 폴더를 엽니다.
4. 압축 안 tools 폴더의 build_sheet.py와 render.py를 업로드해서 기존 파일을 교체합니다.
5. Commit changes를 누릅니다.
6. GitHub Actions의 '편성표 이미지 생성' 작업이 끝날 때까지 기다립니다.
7. https://hhsliving.github.io/schedule/ 를 새로고침합니다.

적용 결과
- 사업부와 팀편성 이미지가 같은 1869 x 1100 규격으로 생성됩니다.
- 두 화면의 요일 행과 그 아래 편성표가 같은 높이에 고정됩니다.
- index.html은 직접 수정할 필요가 없습니다.

주의
- 두 Python 파일은 반드시 함께 교체하세요.
- Actions 완료 전에는 기존 이미지가 계속 보일 수 있습니다.
