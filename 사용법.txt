가전팀 편성표 뷰어
==================

[여는 법]
  GitHub Pages 주소 또는 index.html 을 엽니다. (크롬 권장)
  ※ html 파일과 images 폴더는 항상 같은 위치에 두세요.

[조작]
  사업부 / 팀편성 탭   : 시트 전환 (키보드 1 / 2)
  ‹  ›  버튼           : 주 이동 (키보드 ← / →)
  찾기 드롭다운        : 연도 · 월 · 주차 선택해서 바로 이동
  아래 주차 레일       : 클릭해서 바로 이동 (주황 표식 = 월 교차 주간)
  이미지 클릭          : 원본 크기로 확대 (드래그로 이동, 다시 클릭하거나 ESC 로 축소)
                         ※ 기본 상태에서는 편성표 전체가 스크롤 없이 화면에 맞춰집니다

[현재 등록된 주차]
  2026년 01월 2주 ~ 07월 5주 (총 20주)
  ※ 과거 주차는 비중을 비워둔 상태입니다. 07월 4·5주만 비중이 채워져 있습니다.
  ※ 01~05월과 07월은 원본 편성표 양식이 달라 이미지 가로폭이 다릅니다 (원본 그대로).

[주차 추가하는 법]
  1) images 폴더에 아래 이름으로 png 를 넣습니다.
       dept-2026-08-1.png   (사업부)
       team-2026-08-1.png   (팀편성)
       → 규칙: {dept|team}-{연도}-{월2자리}-{주차}.png

  2) index.html 을 메모장으로 열고, 상단 WEEKS 배열에 한 줄 추가합니다.
       { y: 2026, m: 8, w: 1, span: "08/03 ~ 08/09", cross: false },
       → cross 는 월이 바뀌는 주간이면 true

  저장하고 새로고침하면 끝입니다.
